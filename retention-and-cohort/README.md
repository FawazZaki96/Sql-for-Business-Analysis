# Retention & Cohort Analysis (SQL)

## Business Questions
- How many customers return after their first purchase?
- What is the retention rate by cohort (first purchase month)?
- How does customer activity decay over time?
- Which cohorts are the most valuable and loyal?

## KPIs
- Cohort Size
- Monthly Retention Rate
- Active Customers by Cohort Month
- Churn Proxy

## Techniques
- Cohort table using first purchase date
- Self-joins and CTEs
- Window functions
- Time difference calculations
