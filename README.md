# SQL for Business Analysis

This repository contains business-focused SQL queries used for KPI reporting, growth analysis, customer segmentation, and decision support.

## Objectives
- Demonstrate practical SQL skills for Business Analyst and Analytics Consultant roles
- Translate business questions into structured SQL logic
- Apply window functions, CTEs, and aggregations for management reporting

## Modules

### 1. Revenue & Growth
MoM growth, cumulative revenue, YTD, trend analysis using window functions.

### 2. Customer Analytics
New vs returning customers, AOV, Pareto (80/20), lifetime value.

### 3. Product & Category Performance
Top SKUs, category contribution, mix analysis.

### 4. Marketing & Funnel
Campaign performance, before/after analysis, conversion metrics.

### 5. Retention & Cohort
Cohort tables, retention curves, churn proxies using SQL.

## Dataset
Queries are designed using the **Online Retail (E-commerce) Transaction Dataset** and can be adapted to any transactional business database.

## Tools & Techniques
- SQL (CTEs, Window Functions, Aggregations)
- Business KPI Design
- Time-Series & Cohort Analysis
